# 🚀 AI Job Application Agent (AIAA)
### *Your Intelligent Career Companion*

AIAA is a cutting-edge mobile application built with **Flutter** that leverages the power of **Google Gemini AI** to revolutionize your job search. From analyzing your CV against job descriptions to generating professional cover letters, AIAA is designed to give you a competitive edge.

---

## ✨ Key Features

- **📄 CV Management**: Securely upload and manage multiple versions of your CV.
- **🔍 Smart Analysis**: Instantly compare your CV with any job description to find skill gaps and match percentages.
- **✍️ AI Cover Letters**: Generate tailored, professional cover letters in seconds.
- **📊 Application Tracker**: Keep track of your job hunt status (Saved, Applied, Interview, Offer, Rejected).
- **💾 Analysis History**: Access previous AI insights anytime.
- **🔒 Secure Auth**: Personalized experience with secure login and registration.

---

## 🛠️ Tech Stack

- **Frontend**: Flutter (Dart)
- **State Management**: BLoC / Cubit
- **Navigation**: GoRouter
- **Networking**: Dio
- **Storage**: Flutter Secure Storage

## 🚀 Getting Started

### Prerequisites
- Flutter SDK (latest version)
- Android Studio / VS Code
- A running instance of the AIAA Backend

### Installation
1. Clone the repository
2. Install dependencies:
   ```bash
   flutter pub get
   ```
3. Create a `.env` file in the root directory:
   ```env
   BASE_URL=https://your-api-url.com
   ```
4. Run the app:
   ```bash
   flutter run
   ```

## 📂 Project Structure
- `lib/core`: Routing, constants, and theme.
- `lib/features`: Functional modules (Auth, CV, Job, Analysis).
- `lib/shared`: Reusable widgets and utilities.
